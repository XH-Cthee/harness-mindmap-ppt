TWAIN 驱动层功能划分

▎基础功能（必须实现，否则无法完成基本扫描）

功能项 功能描述 对应 Capability 必备程度 适用机型
颜色模式设置 支持黑白、灰度、RGB 彩色三种像素类型切换 ICAP_PIXELTYPEICAP_BITDEPTH 核心必备 所有机型
分辨率设置 X/Y 轴 DPI 可配置，覆盖 75~1200 dpi 常用档位 ICAP_XRESOLUTIONICAP_YRESOLUTION 核心必备 所有机型
页面尺寸设置 支持 A4、Letter、Legal 等标准尺寸及自定义尺寸 ICAP_SUPPORTEDSIZESICAP_FRAMES 核心必备 所有机型
数据传输机制 支持 Native / Memory / File 至少一种传输模式 ICAP_XFERMECHCAP_XFERCOUNT 核心必备 所有机型
设备在线检测 返回扫描仪是否就绪可用 CAP_DEVICEONLINE 核心必备 所有机型
无 UI 静默扫描 支持禁用数据源 UI，由上层软件完全控制 CAP_UICONTROLLABLE 核心必备 所有机型
图像格式协商 支持 BMP/JPEG/TIFF/PNG/PDF 等输出格式声明 ICAP_IMAGEFILEFORMAT 基础必备 所有机型

自动边框检测 自动识别文档边缘并裁切空白 ICAP_AUTOMATICBORDERDETECTIONICAP_AUTOSIZE 拓展功能 高端机型

自动旋转 根据内容方向自动旋转图像至正向 ICAP_AUTOMATICROTATE 拓展功能 高端机型
条码检测 扫描时自动识别一维/二维条码 ICAP_BARCODEDETECTIONENABLEDICAP_SUPPORTEDBARCODETYPES 高级选配 专业机型
补丁码检测 识别 Patch Code 分隔页自动拆分文档 ICAP_PATCHCODEDETECTIONENABLED 高级选配 专业机型
图像压缩 JPEG、CCITT G3/G4 等压缩算法支持 ICAP_COMPRESSIONICAP_JPEGQUALITY 拓展功能 所有机型
背书器/打印器 扫描后在原稿背面打印页码或标记 CAP_ENDORSERCAP_PRINTER 高级选配 专业机型
自动颜色检测 自动判断页面彩色/黑白并切换模式 ICAP_AUTOMATICCOLORENABLED 拓展功能 高端机型
伽马校正 调整图像伽马曲线优化显示效果 ICAP_GAMMA 可选 所有机型

测试用例映射建议

TWAIN 驱动层测试重点

基础功能必测：颜色模式、分辨率、页面尺寸、传输机制、设备在线检测、无 UI 扫描、ADF 进纸控制、纸张检测

拓展功能选测：双面扫描、重张检测、自动纠偏、自动裁边、空白页剔除、条码检测、自动颜色检测

兼容性测试：与主流上层软件（Photoshop、Acrobat、PaperStream）的 TWAIN 接口兼容性

上层软件测试重点

基础功能必测：设备管理、扫描控制、参数配置、图像预览、文件输出、异常处理

拓展功能选测：OCR 识别精度、图像处理效果（纠偏/裁边）、条码识别率、云服务连接稳定性、批量任务性能

用户体验测试：界面响应速度、大文件处理性能、多页文档操作流畅度

测试阶段 TWAIN 驱动测试项 上层软件测试项 优先级
P0 冒烟测试 设备发现、基础扫描、数据传输 软件启动、设备连接、单页扫描保存 最高
P1 基础功能 颜色/分辨率/尺寸设置、ADF 进纸、多页连续扫 参数配置、预览编辑、多格式输出、异常提示 高
P2 拓展功能 双面扫描、重张检测、自动纠偏、条码检测 OCR、图像处理、云服务、批量任务 中
P3 高级功能 补丁码、背书器、自动颜色检测 智能分类、索引提取、系统集成、安全合规 低
P4 兼容性 多软件 TWAIN 兼容性、协议合规性 多设备支持、跨平台稳定性 可选
